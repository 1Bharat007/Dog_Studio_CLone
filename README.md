## 🎯 About This Project

This is a **frontend clone** of the award-winning DogStudio creative agency website. The project focuses on replicating the premium feel of the original — smooth scroll animations, cursor effects, and immersive page transitions.

> **Purpose**: A learning project to master advanced CSS animations, GSAP, and JavaScript DOM manipulation techniques used by top-tier creative agencies.

---

## ✨ What's Cloned

| Feature | Implementation |
|---|---|
| 🎬 **Smooth page transitions** | GSAP + custom easing curves |
| 🖱️ **Custom cursor** | JavaScript mouse tracking |
| 📜 **Smooth scroll** | Locomotive Scroll / custom implementation |
| 🎨 **Hover animations** | CSS transforms + GSAP timelines |
| 📐 **Responsive layout** | CSS Grid + Flexbox |

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/1Bharat007/Dog_Studio_CLone.git
cd Dog_Studio_CLone

# Open in browser (no build step needed)
# Use Live Server extension in VS Code, or:
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000)
